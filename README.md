# 🎤 Speech-to-Text and Text-to-Speech Web Application

A Python and Django-based web application that converts speech into text and text into speech using audio processing techniques.

---

# 🚀 Project Overview

This project provides both:

- Speech-to-Text Conversion
- Text-to-Speech Conversion

The application allows users to:
- Convert spoken audio into text
- Generate speech audio from text input
- Interact through a simple web interface

---

# 🛠️ Technologies Used

- Python
- Django
- SpeechRecognition
- gTTS (Google Text-to-Speech)
- PyAudio
- HTML/CSS

# ⚙️ Working Process

## Speech-to-Text
- Captures audio input
- Processes speech signals
- Converts speech into readable text

## Text-to-Speech
- Accepts text input
- Converts text into spoken audio
- Generates voice output

# ▶️ How to Run the Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/dishakataria15/speech-to-text-and-text-to-speech.git
```

---

## 2️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Run Django Server

```bash
python manage.py runserver
```

---

## 4️⃣ Open in Browser

```text
http://127.0.0.1:8000/
```
